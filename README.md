# Проект API Yatube

## Описание
REST API для социальной сети Yatube. Позволяет пользователям публиковать посты, оставлять комментарии, просматривать тематические группы и подписываться на других авторов. API поддерживает пагинацию и поиск. Аутентификация реализована по JWT-токенам.

## Технологии
- Python 3.9+
- Django 3.2
- Django REST Framework
- Djoser (JWT)
- SQLite3

## Как запустить проект:
1. Клонировать репозиторий и перейти в папку.
2. Создать и активировать виртуальное окружение.
3. Установить зависимости: `pip install -r requirements.txt`
4. Выполнить миграции: `python manage.py migrate`
5. Запустить сервер: `python manage.py runserver`
